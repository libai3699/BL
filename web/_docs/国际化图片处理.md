# 国际化图片处理

## 公用图片

如果某个图片可以预测到就算换皮肤、换套系都还是用这个图片，就用这个策略
使用方式：

1. 图片放在 src/static/common 中对应语言下面
2. 要使用的页面，头部加上：
   import { useI18n } from 'vue-i18n';
   import { imageMapCommon } from '@/static/images';
   const { locale } = useI18n();
3. 需要使用到的地方：

- style：
  :style="{
  background: `url(${imageMapCommon[locale]['invite-bg']}) no-repeat 0 0 / 750rpx 2044rpx`
  }"
- image：
  <image :src="imageMapCommon[locale]['invite-bg']" />

## 套系图片

如果某个图片在每个套系(series)中都不同，用这个策略
使用方式

1. 先定位图片文件夹: static/images/series
2. 选择套系文件夹，比如 default
3. 可以在某个子文件夹中新建语言文件夹，也可以直接在 default下建
4. 比如 advisor文件夹 中，建立三个语言文件夹: zh_CN, en_US, es_MX
5. 把图片放到语言文件夹中
6. 其他套系如果也需要，也依葫芦画瓢
7. 要使用的页面，头部加上：
   import { useI18n } from 'vue-i18n';
   import images from '@/static/images';
   const { locale } = useI18n();
8. 使用示例
   以图片在 static/images/series/default/advisor/(zh_CN, en_US, es_MX)/invite-bg.png 示例

- style：
  :style="{
  background: `url(${images.advisor[locale]['invite-bg']}) no-repeat 0 0 / 750rpx 2044rpx`
  }"
- image：
  <image :src="images.advisor[locale]['invite-bg']" />
